## 📞 Troubleshooting

### Frontend not loading?
- Check: `http://localhost:5174/`
- Try: Clear cache (Ctrl+Shift+Delete)
- Check: Console for errors (F12)

### Images not showing?
- Check: Files in `frontend/public/assets/images/`
- Check: Filenames are `car.png` and `dap-logo.png`
- Check: Console for 404 errors

### API calls failing?
- Check: Django running on port 8000
- Check: CORS enabled in Django
- Check: Endpoint URLs correct

### Port already in use?
- Use: `npm run dev -- --port 5175`
